用法:  
        Windows_Notification_Push_Open_Platform.exe [选项]  
  
选项:  
        --action : 在按钮中设置操作  
        *--aumi : 设置应用用户模型 ID  
        *--name : 设置显示应用名称  
        *--id : 设置应用ID  
        --time : 设置过期时间  
        --text01 : 设置通知的第一行文本  
        --text02 : 设置通知的第二行文本  
        --text03 : 设置通知的第三行文本  
        --help : 打印帮助说明  
        --image-path : 设置图像路径  
        --only-create-shortcut : 仅创建应用的快捷方式  
        --audio-state : 设置通知音频的播放模式：单次 = 0，无 = 1，多次循环 = 2  
        --attribute : 设置通知的注释
